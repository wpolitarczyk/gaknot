# Generalized Algebraic Knot Invariants

[![CI Tests](https://github.com/wpolitarczyk/gaknot/actions/workflows/test-package.yml/badge.svg)](https://github.com/wpolitarczyk/gaknot/actions/workflows/test-package.yml)
[![pipeline status](https://gitlab.mimuw.edu.pl/wpolitarczyk/signature-function/badges/master/pipeline.svg)](https://gitlab.mimuw.edu.pl/wpolitarczyk/signature-function/-/commits/master)

This project allows calculating signature invariants for generalized algebraic knots (`GA-knots`).
Generalized algebraic knots are knots constructed as connected sums of positive iterated torus knots and their concordance inverses.
It was created as part of the proof for the main lemma from the paper **"On the slice genus of generalized algebraic knots"** (Maria Marchwicka and Wojciech Politarczyk) ([arXiv:2107.11299](https://arxiv.org/abs/2107.11299)).

## Project Structure

The project follows a standard Python `src/` layout:

* **`src/gaknot/`**: Contains the core source code (.sage files are the source of truth).
* **`tests/`**: Contains standard Python test files using `pytest`.
* **`pyproject.toml`**: Project metadata and dependency configuration.
* **`Makefile`**: Automation for building, installation, testing, and cleanup.

## Requirements

* **SageMath**: This project relies on the SageMath library. Ensure you have a working installation of SageMath (e.g., via Conda or a native install).
* **pytest**: Required for running the test suite.

## Installation

### For Developers (Recommended)
The package uses a build step to preparse `.sage` files into `.py` files. Install the package in "editable" mode:

```bash
# Ensure your sage environment is active (e.g., conda activate sage_env)
make install
```

This will run the preparse step and then execute `pip install -e .`.

## Usage

Once installed, you can use the library with standard Python imports.

### 1. Working with Generalized Algebraic Knots
The easiest way to work with the package is using the `GeneralizedAlgebraicKnot` class.

```python
from gaknot import GeneralizedAlgebraicKnot

# Define T(2,3)
knot1 = GeneralizedAlgebraicKnot([(1, [(2, 3)])])

# Define an iterated torus knot: (6,5)-cable of T(2,3)
knot_iter = GeneralizedAlgebraicKnot([(1, [(2, 3), (6, 5)])])

# Operations: Connected sum (+) and Concordance inverse (-)
sum_knot = knot1 + knot_iter
inverse_knot = -knot1

# Compute the signature function
sig_func = sum_knot.signature()
print(f"Is the signature zero everywhere? {sig_func.is_zero_everywhere()}")
```

### 2. Branched Cover Homology
The `BranchedCoverHomology` class computes the first homology group $H_1(\Sigma_N(K))$.

```python
from gaknot import BranchedCoverHomology

# Compute H_1 of the 2-fold branched cover of the trefoil
h1 = BranchedCoverHomology(knot1, 2)
print(h1) # Output: (Z/3Z)[T(2,3)]
```

### 3. Visualization
`SignatureFunction` objects can be plotted using the `SignaturePloter`.

```python
from gaknot import SignaturePloter

# Plot the step function
SignaturePloter.plot(sig_func, title="Signature Function")
```

## Development and Testing

### Building the Package
To preparse all `.sage` files into `.py` files:


Run the standard `pytest` suite:
```bash
make test
```
